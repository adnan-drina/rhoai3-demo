# Legacy Implementation Backup

This directory contains the previous implementation moved aside for the RHOAI
demo reimplementation.

Tracked legacy content is preserved under its original relative paths, including:

- `gitops/`
- `scripts/`
- `steps/`
- root `README.md`
- promoted operational docs from `docs/`
- generated architecture SVGs from `docs/assets/architecture/`

The `local-untracked/` directory, when present, contains local workspace
residual files that were not tracked by Git. It is intentionally ignored and is
not part of the preserved implementation record.

Do not run legacy scripts from this backup unless explicitly restoring or
inspecting the old implementation.
